## Description

# Cat Facts

Modelling the cat facts REST API using nest js.

[Reference Link](https://github.com/alexwohlbruck/cat-facts)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
